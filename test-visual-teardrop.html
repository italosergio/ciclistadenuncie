❌ ANTES — <rect rx=12> rotacionado (diamante)

⚠️ <rect> com rx=12 + rotate(-45°) forma um losango/diamante arredondado — NÃO é uma teardrop de verdade. Os cantos superiores e inferiores ficam com curvatura igual.

✅ DEPOIS — Path SVG com 3 arcos (teardrop verdadeira)

✅ Path com M 14,2 A ... A ... L 2,26 Z — 3 arcos circulares (topo, direita, base) + fechamento reto no ponto inferior. Com rotate(-45°) vira uma gota/teardrop de verdade, igual ao cluster marker.
